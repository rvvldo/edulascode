# Background Music

Letakkan file musik background di folder ini dengan nama `background-music.mp3`

## Format yang Didukung:
- MP3
- WAV
- OGG

## Rekomendasi:
- Durasi: 2-5 menit (akan di-loop)
- Ukuran: < 5MB untuk performa optimal
- Volume: Sudah dinormalisasi (tidak terlalu keras)

## Contoh Musik yang Cocok:
- Ambient music
- Lo-fi beats
- Nature sounds
- Instrumental music

## Cara Menambahkan:
1. Siapkan file musik Anda
2. Rename menjadi `background-music.mp3`
3. Copy ke folder `public/mp3/`
4. Refresh aplikasi
